# AI Digital Twin - Parveen Kumar Das
An advanced conversational Agentic AI platform designed to replicate my professional persona, skills, and projects. Built natively using the **Google GenAI SDK** and powered by the **Gemini 2.5 Flash** architecture, this digital twin acts as a 24/7 interactive portfolio.

## Key Capabilities

* **Autonomous Tool Calling:** The agent doesn't just talk; it takes action. It uses live API integration and system operations to handle real-time queries seamlessly.
* **Contextual Awareness:** Equipped with a dedicated system instruction layer that guides its personality, tone, and data retrieval as my digital representative.
* **Gradio User Interface:** Wrapped in a clean, intuitive, and production-ready chat dashboard for instant interaction.

## Integrated Native Tools

1. **Live Weather Tracking:** Fetches real-time temperature updates using dynamic API requests.
2. **System Clock/Date Analytics:** Dynamically calculates relative dates (today, tomorrow, yesterday) based on the system's live clock.
3. **Dynamic File Generation:** Generates and exports a fresh text-based resume file (`Parveen_Resume.txt`) directly onto the sidebar on user demand.

## Tech Stack
- **Core LLM:** Google Gemini 2.5 Flash
- **Framework/SDK:** Google GenAI Native SDK
- **Interface:** Gradio UI Components
- **Language:** Python 3.x
